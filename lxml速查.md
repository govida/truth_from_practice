# lxml速查

| 表达式            | 描述                         |
| ----------------- | ---------------------------- |
| .                 | 当前节点                     |
| ..                | 当前节点的父节点             |
| /                 | 当前节点的直接子节点         |
| //                | 当前节点的子孙节点           |
| @                 | 当前节点的所有子节点         |
| [@attrib='value'] | 选取给定属性具有给定值的元素 |
|                   |                              |
|                   |                              |
